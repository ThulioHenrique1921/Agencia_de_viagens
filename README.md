# Agencia_de_viagens
Esse é um pequeno projeto de HTML + CSS para paraticar alguns principios importantes.
O objetivo dessa página é criar uma plataforma de captação de clientes para uma agência de viagem que possua un fornulário para obter orçamentos. 
Para sua criação foram aplicados conceitos de formulários html, flexbox, classes e subclassses css, media queries e mobile first.
